## Tech Stack

TypeScript, React, Tailwind, Font Awesome Icons

## Run Locally

Clone the project

```bash
  git clone git@github.com:denysdev1/react-tailwind-ts-storyteller.git
```

Go to the project directory

```bash
  cd react-tailwind-ts-storyteller
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
